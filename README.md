# Heart Disease Prediction Using Machine Learning

## Project Description

This project is based on the IEEE conference paper:

**Comparative Analysis of Classification Models for Heart Disease Prediction Using Machine Learning**

The goal of the project is to compare machine learning classification models for predicting heart disease using clinical patient data.

## Models

The following classification models will be evaluated:

- Logistic Regression
- K-Nearest Neighbors
- Support Vector Machine
- Random Forest

## Project Extension

In addition to reproducing the methodology described in the paper, the project includes:

- 5-Fold Cross Validation
- Hyperparameter Tuning using GridSearchCV
- Comparison of model performance before and after tuning

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Dataset

The project uses the Heart Disease Dataset referenced in the selected paper and available through Kaggle.
